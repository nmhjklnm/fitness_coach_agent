## 💡 核心特性

**AI 工作模式**：制定计划 → 执行记录 → 动态调整 → 定期复盘  
**训练策略**：练六休一（周二-周日训练/周一休），无氧/有氧交替，≤45分钟/次，每2周负重+5-10%  
**饮食策略**：极限期1250-1350 kcal/日（3周）→ 回馈期1800-2000 kcal/日（1周），蛋白≥150g/天  
**监测体系**：体重/周、体脂/月、围度/周、RPE/次
